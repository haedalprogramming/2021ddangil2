# 딴길학교 2 첫번째 과제
ASCII Art에 대해 알아보고, 원하는 이미지를 출력해보자!  
`ASCII Art`를 출력한 화면을 컴퓨터에 이미지로 저장합니다.  

# Cows
```

     `\------(oo)
       ||    (__)
       ||w--||     \|/
   \|/
```
